How to get Etisalat configuration settings

Posted by Lourie Helzer on Sunday, June 30, 2024

Want to have Internet on your phone with you? Get Etisalat configuration message and tune your phone as due to be able to browse the Web and find whatever you need, communicate with people and so on. Don’t know how to get the message? Keep reading and follow the step-by-step guide.

Reaching the Internet from your pocket

Having the access to the Internet everywhere and every minute is a must nowadays. Checking social networks, uploading pictures, commenting others’ photos, downloading music and watching videos, and so on, this is the life of a 21st century person. The only way to have it is get the access via the cell phone and a good provider. In Nigeria in particular, one of these good providers is Etisalat.

How to get Etisalat configuration settings automatically

It’s very simple if you want to start using the Internet and configure your access via the message service. It’s a good choice if you are not very knowledgeable about the way all those applications in mobile phones work or just don’t want to waste your time, entering a lot of details. Open your text message menu, type SETTINGS and send the text to 790. This is, namely, the Etisalat configuration settings code.

few seconds after you send the request, you will get Etisalat configuration message. You only need to save it, then, select the Activate All option, and that’s it. You can start browsing the Internet.

However, you need to remember that automatic settings don’t always work at once. In case something goes wrong and you can’t start using the Internet after the automatic configuration or so, try doing it manually. There’s nothing complicated in it but try to do everything accurately in order to skip or mix no symbols; otherwise, you will have to start the process from the very beginning.

Here’s what you need to do to configure WAP:

  • Profile/ Access Point Name - type Etisalat WAP
  • APN/ Access Point – type etisalat
  • Username/Login – leave it blank
  • Password – leave it blank
  • IP/Proxy – type 010.071.170.005
  • Port – type 8080
  • Home Page - type http://mobile.etisalat.com.ng

Some experienced people say that if you leave the IP and Port boxes empty, it will make no big difference. Yet, if you have the data, it’s probably better to fill it in just in case. It will give you more chances that everything will work correctly and you won’t need to turn to specialists to have your phone tuned properly.

Etisalat Internet Settings:

  • Account Name – type Etisalat Internet
  • Username – leave it blank
  • Password – leave it blank
  • Access Point Name (APN) – type etisalat
  • Home Page – type http://mobile.etisalat

Etisalat configuration settings for Android

If you have a smartphone powered by Android, you may need an additional set of configuration settings. Here’s what you need to do to have the access to the Etisalat Internet from your Android device:

Go to the menu SETTINGS, open WIRELESS AND NETWORKS and then choose NETWORKS.

Here you need to select MOBILE NETWORKS, then, GSM/UMTS OPTIONS, and finally, ACCESS POINT NAMES.

When you have reached it, select the line ADD NEW APN and fill in all the boxes with the following parameters:

  • Name– type Etisalat
  • APN – type Etisalat
  • Proxy – type 10.71.170.5
  • Username – type wap
  • Password – type wap
  • Authentication Type – type PAP or CHAP

Now, everything should be working well!
